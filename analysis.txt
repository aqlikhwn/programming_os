                    CLASS SCHEDULING ALGORITHM ANALYSIS

1. ALGORITHM SELECTION
--------------------------------------------------------------------------------
   We chose SJF (Shortest Job First - Non-Preemptive) for comparison with FCFS 
   because class scheduling operates on a non-preemptive basis - once a class 
   begins, it cannot be interrupted.

   Available Algorithms:
   +------------------+-------------+--------------+
   | Algorithm        | Type        | Usable?      |
   +------------------+-------------+--------------+
   | Round-Robin      | Preemptive  | NO           |
   | SRTF             | Preemptive  | NO           |
   | SJF              | Non-Preempt | YES          |
   | Priority         | Non-Preempt | YES          |
   | FCFS             | Non-Preempt | YES          |
   +------------------+-------------+--------------+

   SJF is designed to reduce average waiting time by prioritizing the 
   completion of shorter tasks first.


2. INPUT DATA
--------------------------------------------------------------------------------
   The following courses were used for testing:

   +-------------+----------------+----------+--------------+
   | Course Code | Duration (hrs) | Priority | Arrival Time |
   +-------------+----------------+----------+--------------+
   | CSC2201     | 3              | 2        | 1            |
   | CSC3401     | 2              | 3        | 2            |
   | CSC1103     | 1              | 1        | 3            |
   +-------------+----------------+----------+--------------+


3. OUTPUT RESULTS
--------------------------------------------------------------------------------

   a) FCFS (First Come First Serve)
   ---------------------------------
   Scheduling Order: CSC2201 -> CSC3401 -> CSC1103

   +-------------+----------+------------+--------------+-----------------+
   | Course Code | Duration | Completion | Waiting Time | Turnaround Time |
   +-------------+----------+------------+--------------+-----------------+
   | CSC2201     | 3        | 3          | 0            | 3               |
   | CSC3401     | 2        | 5          | 3            | 5               |
   | CSC1103     | 1        | 6          | 5            | 6               |
   +-------------+----------+------------+--------------+-----------------+

   Average Waiting Time:    2.67 hours
   Average Turnaround Time: 4.67 hours


   b) SJF (Shortest Job First - Non-Preemptive)
   ---------------------------------------------
   Scheduling Order: CSC1103 -> CSC3401 -> CSC2201

   +-------------+----------+------------+--------------+-----------------+
   | Course Code | Duration | Completion | Waiting Time | Turnaround Time |
   +-------------+----------+------------+--------------+-----------------+
   | CSC1103     | 1        | 1          | 0            | 1               |
   | CSC3401     | 2        | 3          | 1            | 3               |
   | CSC2201     | 3        | 6          | 3            | 6               |
   +-------------+----------+------------+--------------+-----------------+

   Average Waiting Time:    1.33 hours
   Average Turnaround Time: 3.33 hours


4. COMPARISON SUMMARY
--------------------------------------------------------------------------------

   +---------------------+------------+------------+-------------+
   | Metric              | FCFS       | SJF        | Improvement |
   +---------------------+------------+------------+-------------+
   | Avg Waiting Time    | 2.67 hours | 1.33 hours | 50% better  |
   | Avg Turnaround Time | 4.67 hours | 3.33 hours | 29% better  |
   +---------------------+------------+------------+-------------+


5. CONCLUSION
--------------------------------------------------------------------------------
   From the results obtained, the SJF algorithm has better performance compared 
   to FCFS for class scheduling:

   WHY SJF IS BETTER:
   - Minimizes average waiting time (1.33 hrs vs 2.67 hrs)
   - Lower average turnaround time (3.33 hrs vs 4.67 hrs)
   - By prioritizing shorter classes, more courses complete earlier
   - Students waiting for shorter classes don't wait behind long ones

   WHY NOT OTHER ALGORITHMS:
   - Round-Robin: Preemptive - cannot interrupt classes
   - SRTF: Preemptive - would require stopping a class mid-lecture
   - Priority: Valid option, but may cause longer waits if high-priority 
               courses have long durations

   WHY FCFS IS LESS EFFICIENT:
   - Simple and fair, but results in longer waiting times
   - Long classes scheduled first delay all subsequent classes

   FINAL RECOMMENDATION:
   SJF (Non-Preemptive) is the recommended algorithm for class scheduling 
   because it significantly reduces average waiting time while respecting 
   the constraint that classes cannot be interrupted.
